# نظام إدارة الطلاب

## وصف المشروع
نظام ويب ديناميكي كامل لإدارة بيانات الطلاب، يحتوي على لوحة تحكم للأدمن ودعم رفع الصور.

## فكرة المشروع
تطوير نظام إداري يساعد الكلية في تسجيل ومتابعة وإدارة بيانات الطلاب بطريقة منظمة وآمنة.

## المميزات المنفذة
- لوحة تحكم كاملة للأدمن
- CRUD كامل للطلاب (إضافة - تعديل - حذف - عرض)
- رفع صور الطلاب
- نظام تسجيل دخول وتسجيل خروج
- توزيع الصلاحيات (أدمن / طالب)
- استخدام Sessions و Prepared Statements

## طريقة التشغيل

1. استورد ملف `database.sql` في phpMyAdmin (قاعدة البيانات: `students_system`)
2. ضع مجلد المشروع داخل `htdocs` (XAMPP)
3. افتح المتصفح واذهب إلى:  
   `http://localhost/student-system`
